# EthnoSphere

An interactive 3D globe for exploring traditional musical instruments from around the world.

<br>

## Features

- **3D Globe** — Navigate an interactive Earth with atmosphere and day/night cycle
- **85+ Instruments** — From Japanese Koto to Irish Bodhrán
- **Live Audio** — Authentic sounds via Freesound API
- **Modern UI** — Clean, minimal interface with smooth animations

<br>

## Quick Start

```bash
git clone https://github.com/trekar99/Etnosphere.git
cd Etnosphere
npm install
npm run dev
```

Open http://localhost:5173

<br>
